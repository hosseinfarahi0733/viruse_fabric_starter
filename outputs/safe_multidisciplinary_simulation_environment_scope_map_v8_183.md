## Discipline role map

| Discipline | Safe input to the model | Role in targeted-looking behavior |
|---|---|---|
| Biology | Abstract agent state, abstract entry state, abstract replication state, abstract mutation state | Shows that different agent-state and compartment-state constraints can make behavior appear selective without modeling a real pathogen. |
| Chemistry / Biochemistry | Unitless abstract affinity score and stability score | Shows that compatibility gates can create apparent preference, without naming real molecular targets or giving real binding parameters. |
| Physics | Random walk, diffusion field, contact rate, abstract barrier, abstract dose field | Shows that reaching a location matters even when compatibility exists. Contact and transport can mimic selectivity. |
| Mathematics | ODE/PDE templates, stochastic processes, graph dynamics, Bayesian uncertainty, constrained optimization for toy variables only | Provides the computational grammar of the simulation while keeping all variables synthetic and non-operational. |
| Geometry / Topology | Abstract tissue-like lattice, compartment graph, distance field, path constraint | Shows that shape, distance, and connectivity can generate apparent targeting. The agent moves through geometry, not through a magical intent detector. |
| Immunology | Abstract defense field, suppression score, delay term, clearance state | Shows that some compartments can suppress the agent before expansion, making other compartments appear favored. |
| Evolution | Toy variant ensemble, abstract fitness landscape, mutation-noise term | Shows how variation plus selection pressure can create non-intentional adaptation-like patterns. |
| Neuroscience | Abstract barrier compartment and protected-region graph | Represents barrier crossing as a generic compartment problem, not a real CNS invasion pathway. |
| Complexity / Systems | Feedback loops, multiscale coupling, emergent behavior, nonlinear thresholds | Explains why simple local rules can look intelligent or targeted when accumulated across scales. |
| Programming / Simulation Engineering | Safe simulator architecture, synthetic fixtures, deterministic seeds, test harnesses, boundary checks | Turns the map into executable toy simulations without importing operational biological content. |

## Core modeling principle

The simulation must represent targeted-looking behavior as an emergent result of constraints.

It must not represent targeting as an engineered biological design objective.

Safe interpretation:

- Agent behavior appears selective because compartments differ.
- Compartments differ because abstract constraints differ.
- Constraints include transport, compatibility, defense, geometry, stochasticity, and feedback.
- No real-world pathogen tuning is modeled.

## Safe toy simulation sketch

A safe simulator can ask:

Given an abstract graph of compartments, do simple constraints make one compartment appear preferred?

Example safe components:

- Graph nodes: synthetic compartments.
- Graph edges: synthetic movement links.
- Node fields: compatibility, defense, transport resistance.
- Agent state: abstract active/inactive state.
- Observation: aggregate occupancy or activity score.
- Output: targeted-looking pattern score.

The output should be interpreted only as a toy demonstration of emergence.
