## Abstract

VF-H2 studies whether a persistent memory-ledger condition can produce a positive toy signal relative to a no-persistent-memory null control in a safe abstract graph-like model. Prior internal toy checks showed consistent positive `ledger_effect_size` behavior across limited simulation, preregistered larger robustness checks, artifact-resistance and ablation checks, and independent implementation-level replication.

This note integrates related-work context for graph dynamical systems, Memory and Non-Markovian network processes, temporal networks, memory-augmented rule systems, reservoir-style memory analogies, null models, and robustness methodology. The citations are used only for bounded background, terminology support, conceptual positioning, and methodological framing. They do not validate VF-H2.

## 1. Introduction

The working question behind VF-H2 is whether persistent memory can change propagation-like outcomes in a finite safe abstract toy system. The model is intentionally abstract. It does not use real biological datasets, real pathogen models, receptor parameters, operational targeting, wet-lab procedures, infectivity optimization, immune evasion optimization, or host range prediction.
