## 1. Graph Dynamical Systems and Discrete Network Dynamics

VF-H2 is framed as a safe abstract toy model over finite graph-like state spaces. The graph dynamical systems literature provides a background vocabulary for discussing graph-based state updates and discrete networked dynamics [@AledoMartinezValverde2015]. Finite dynamical systems also provide general mathematical background for simulation-oriented discrete systems [@JarrahLaubenbacher2008].

This use is conceptual only. These sources do not prove VF-H2, do not validate `ledger_effect_size`, and do not establish formal equivalence between VF-H2 and any existing graph dynamical-system class.

## 2. Memory and Non-Markovian Processes in Networks

VF-H2 uses a persistent memory ledger to distinguish memory-bearing dynamics from no-persistent-memory null controls. Temporal-network and higher-order network literatures provide conceptual background for memory, path dependence, and history-sensitive network representations [@WilliamsLacasaMillanLatora2022; @LambiotteRosvallScholtes2019].

This use is limited to conceptual positioning. These sources do not validate the persistent memory ledger, do not validate the positive toy signal, and do not establish external or empirical support for VF-H2.

## 3. Temporal Networks and Multi-Time Indexing

The three-time coordinate in VF-H2 separates local update time, ledger accumulation time, and projection or observation time. Temporal-network literature provides a cautious background for thinking about time-indexed network processes and dynamic graph structures [@HolmeSaramaki2012].

This use does not validate the VF-H2 three-time coordinate. It only helps position the notation near broader time-indexed network concepts.

## 4. Memory-Augmented Automata and Rule-Based Toy Systems

VF-H2 is a rule-based abstract toy system in which persistent memory can affect update outcomes. Work on cellular automata with memory provides a related conceptual background for how memory can alter the dynamics of discrete rule-based systems [@MartinezAdamatzkyAlonsoSanz2013].

This does not imply that VF-H2 is a cellular automaton, nor does it establish formal equivalence to cellular automata with memory.

## 5. Reservoir-Style Memory and Recurrent State Analogies

Reservoir-style systems and echo state networks provide terminology-level background for memory-bearing recurrent state and memory capacity [@Jaeger2007; @BarancokFarkas2014]. VF-H2 is not presented as a reservoir computing model, and the analogy is intentionally limited.

These citations are used only for terminology support and analogy-level positioning. They do not imply that VF-H2 has reservoir-computing memory capacity.
