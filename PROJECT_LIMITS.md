# VF-H2 / Viruse Fabric — Project Limits after v17.4.0

## Current valid claim

```text
v17.4.0 proves the full restricted proof-spine target from natural base interval assumptions,
under fixedness and score-preserving policy routes.
```

## Explicit non-claims

The project does not currently claim:

```text
- full VF-H2 is proved
- hFixed is derived
- score-key preservation is derived from domain dynamics
- identity-like update is the scientific main route
- empirical validation has been completed
- the manuscript is submission-ready
- the model fully captures the intended scientific domain
```

## Scope boundary

Current scope:

```text
restricted formal proof-spine construction
```

Not current scope:

```text
complete scientific theory
complete domain semantics
empirical validation
submission-ready paper
```

## Reviewer-facing statement

```text
The v17.4 development removes a major formal weakness by replacing the old collapsing route with a natural interval route for the full restricted proof-spine target. However, several assumptions remain external. The next phase is assumption-driven: derive, remove, or classify the remaining fundamental assumptions.
```

## Tagging policy

Do not create a new scientific tag unless one of the following happens:

```text
- a fundamental assumption is removed
- a fundamental assumption is derived from a stronger semantic layer
- a fundamental assumption is sharply classified with formal evidence
```

Documentation-only commits should normally remain untagged.
