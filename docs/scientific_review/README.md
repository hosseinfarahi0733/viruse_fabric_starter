# VF-H2 Scientific Review Pack

This directory is intended to give reviewers a compact scientific entry point into the project.

Recommended reading order:

1. `THEORY.md`
2. `MAIN_THEOREM.md`
3. `ASSUMPTIONS.md`
4. `PROOF_DEBT.md`
5. `NEGATIVE_RESULTS.md`
6. `DEPENDENCY_GRAPH.md`
7. `CORE_LEAN_FILES.md`
8. `REVIEWER_QUESTIONS.md`

This documentation should not be treated as a new scientific milestone.

It is an architecture and review scaffold designed to reduce ambiguity and prevent overclaim.
